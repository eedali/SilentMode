# SilentMode

SilentMode is an anonymous social media platform.

## Project Structure

This is a monorepo containing:

- **backend**: Node.js + Express + TypeScript API
- **mobile**: React Native + Expo + TypeScript mobile application
- **shared**: Shared TypeScript types and utilities

## Setup

1.  Install dependencies:
    ```bash
    npm install
    ```

2.  Start Backend:
    ```bash
    npm run dev:backend
    ```

3.  Start Mobile:
    ```bash
    npm run start:mobile
    ```

## Tech Stack

-   **Backend**: Node.js, Express, TypeScript, Prisma, PostgreSQL (implied), Redis, Cloudflare R2
-   **Mobile**: React Native, Expo, TypeScript
-   **Shared**: TypeScript
